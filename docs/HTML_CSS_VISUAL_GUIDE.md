## Key Visual Elements Explained

### 1. Blue Info Banners (ℹ️)
These appear twice:
- At the top: Instructions for creating the question
- In answer key section: Explains multiple answers feature

### 2. Code Editors (Monospace font)
- Gray background (`bg-gray-50`)
- Border around text area
- Line count shown in top-right
- Resizable (students can drag to expand)

### 3. Live Preview Box
```
┌────────────────────────────────────┐
│ 👁 Live Preview           [Hide]   │  ← Toggle button
├────────────────────────────────────┤
│                                    │
│   [Rendered HTML/CSS here]         │  ← iframe with code
│                                    │
└────────────────────────────────────┘
```

### 4. Question Preview at Bottom
Shows exactly what students will see:
- Question number badge (1)
- Question text
- Rendered preview of the code
- Note about student view

---

## Color Scheme

- **Blue** (`bg-blue-50`, `border-blue-200`, `text-blue-800`)
  - Info banners and instructions
  - Interactive elements

- **Gray** (`bg-gray-50`, `border-gray-200`)
  - Code editor backgrounds
  - Section borders
  - Preview container borders

- **Amber/Yellow** (`bg-amber-50`, `border-amber-200`, `text-amber-700`)
  - Warning notes
  - Manual grading notice

- **White** (`bg-white`)
  - Main background
  - Input areas

---

## Spacing & Layout

- Sections use `space-y-4` (1rem / 16px vertical spacing)
- Padding: `p-3` or `p-4` (0.75rem / 12px or 1rem / 16px)
- Text sizes: `text-xs` (12px), `text-sm` (14px), `text-base` (16px)
- Rounded corners: `rounded-md` or `rounded-lg`

## Interactive Elements

### Toggle Preview Button
```
┌─────────────────────────────────┐
│ 👁 Live Preview        [Hide]   │  ← Click to hide
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ 👁 Live Preview        [Show]   │  ← Click to show
└─────────────────────────────────┘
```

### Resizable Textareas
All code editors have `resize-y` class:
- Students can drag bottom edge to make taller
- Cannot resize horizontally
- Useful for longer code

### Line Counters
```
</> HTML Code *              150 lines  ← Updates in real-time
</> CSS Code                  50 lines  ← Updates in real-time
```


## Full Example with Real Content

```
Question Instructions:
"Create a navigation bar with three links: Home, About, Contact. 
The navigation should have a blue background (#007bff) and white text."

HTML Code (shown to students):
<nav>
  <a href="#home">Home</a>
  <a href="#about">About</a>
  <a href="#contact">Contact</a>
</nav>

CSS Code (shown to students):
/* Add your CSS styling here */

Correct Answer (HTML):
<nav style="background:#007bff">
  <a href="#home" style="color:white">Home</a>
  <a href="#about" style="color:white">About</a>
  <a href="#contact" style="color:white">Contact</a>
</nav>

Correct Answer (CSS):
nav {
  background-color: #007bff;
}
nav a {
  color: white;
  padding: 10px;
  text-decoration: none;
}

Alternative Answers:
nav {
  background: #007bff;
  padding: 10px;
}
a {
  color: #fff;
}
---
nav {
  background-color: rgb(0, 123, 255);
}
nav a {
  color: #ffffff;
  display: inline-block;
  padding: 0.5rem 1rem;
}
```
