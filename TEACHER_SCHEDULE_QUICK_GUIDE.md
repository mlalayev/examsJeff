# Teacher Schedule System - Quick Guide

## What You Asked For ✅

1. ✅ **Teacher can name the class** - Yes! Class name field (e.g., "English 10A")
2. ✅ **Add students with names** - Yes! Students have first name AND last name
3. ✅ **See earnings** - Yes! Shows total earnings for odd/even days at the top
4. ✅ **1-hour lessons only** - Yes! Predefined hourly slots (09:00-10:00, 10:00-11:00, etc.)
5. ✅ **No save button** - Yes! Auto-saves automatically after every change

## How Teachers Use It

### Adding a Class

1. Go to Schedule page
2. Choose Odd or Even days tab
3. Click "Add Class"
4. Fill in:
   - **Class Name**: "Mathematics 10A"
   - **Time Slot**: Select from dropdown (09:00-10:00, etc.)
   - **Hourly Rate**: How much you earn (e.g., $50.00)
5. Click "Add Student":
   - Enter **First Name**: "John"
   - Enter **Last Name**: "Doe"
   - Click "Add"
   - Repeat for more students
6. Click "Add Class" - **Automatically saved!**

### What Teachers See

**At the Top:**
- Purple card: "Odd Days Earnings: $150.00 (3 classes)"
- Blue card: "Even Days Earnings: $95.00 (2 classes)"

**For Each Class:**
- Class name (big and bold)
- Time slot badge
- Number of students
- Hourly rate in green
- All student names listed below
- Edit and Delete buttons

## Technical Info

- **Location**: Dashboard → Schedule (in sidebar)
- **Auto-save**: Changes save automatically in background
- **Database**: All data persists in PostgreSQL
- **Access**: Only teachers can see their own schedule
